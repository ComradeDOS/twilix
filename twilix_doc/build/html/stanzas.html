Twilix.Stanzas Documentation¶

Module discribes stanza-class and basic types of used stanzas. There are Message, Presence, Iq classes inherit from Stanza.

class twilix.stanzas.ErrorStanza(*args, **kwargs)¶

Stanza-inheritor class for any errors. Contains fields corresponding to the protocol.

Attributes:
error – element node with Error element.
class twilix.stanzas.Iq(**kwargs)¶

Stanza-inheritor class that implements an answer-query type mechanism.

clean_id(value)¶

Set correct value for invalid id. Used for validation of id.

clean_type_(value)¶

Filters stanzas by type. Used for validation of type_ field. Raises exception for invalid stanzas.

Raises :ElementParseError
makeResult()¶

Returns result-type Iq stanza

class twilix.stanzas.Presence(*args, **kwargs)¶

Stanza-inheritor class that implements an entity’s presence info. Contains fields corresponding to the protocol.
